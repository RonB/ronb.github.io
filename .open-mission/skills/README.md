# Skills

Repository-owned skills live here. Skills provide bounded operational guidance and do not replace workflow law or the repository governance tree.

- `code-quality-control`: review recently changed implementation code before audit.
- `governance-writer`: maintain the repository governance harness after initialization and before governed Mission work; defaults to autonomous mode and governance domain skills live under `governance-writer/`.
